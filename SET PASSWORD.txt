SET PASSWORD
-----------------------------------------------------------
MSX2 以降
MSXPLAYer では動作しません

[ 機能 ]
パスワードを設定します。

[ 書式 ]
SET PASSWORD <パスワード>

[ 文例 ]
SET PROMPT "PaSsWoRd"

[ 解説 ]
<パスワード>の値は255文字以内です。起動時にパスワードを聞かれ、入力しないと起動しなくなります。ただし [GRAPH]+[STOP] を押しながら起動する事でパスワードを聞かれずに起動する事が可能です。
この設定は MSX 本体では SRAM に記録されます。SET PROMPT・SET TITLE と共有しているため、これらと含めて最後に設定されたものが有効となります。
初期状態は SET TITLE "" として下さい。
