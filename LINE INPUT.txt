LINE INPUT
-----------------------------------------------------------
[ 機能 ]
キーボード・ファイルから入力された1行のデータを変数に入れます。

[ 書式 ]
LINE INPUT { [<プロンプト>;] | #<ファイル番号> }<変数名>

[ 文例 ]
LINE INPUT "anata no onamae nanntehno > ";N$

[ 解説 ]
<ファイル番号> を入れない場合はキーボードからの入力を受け付けます。
<プロンプト> があればそれを表示し、カーソルを表示する事で入力を促します。入力された内容が <変数名> に入ります。
INPUT と異なり ? が表示されません。また <変数名> は１つだけで複数の <変数名> は指定できません。
グラフィック画面（SCREEN 2〜8・10〜12）の場合はテキスト画面に戻ります。
なお、入力で空白や , を含む場合は " で囲む事で受け付けされます。

<ファイル番号> が入っている場合は、そのファイルから入力を受け付けます。
<ファイル番号> のファイルは OPEN により INPUT モードで開かれている必要があります。
ファイル中では、空白・改行・ , 以外の文字がデータの区切りと成ります。また、" で囲まれたものはそれで一つのデータと見なされます。255文字までを一度に読み込みます。

入力として入る文字は最大255文字です。改行は含まれません。
