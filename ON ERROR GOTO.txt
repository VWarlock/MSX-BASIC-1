ON ERROR GOTO
-----------------------------------------------------------
[ 機能 ]
エラー発生時の行番号を定義します。

[ 書式 ]
ON ERROR GOTO <行番号>

[ 文例 ]
ON ERROR GOTO 10000

[ 解説 ]
通常エラーが発生した場合はエラーメッセージを表示して処理を中断しますが、それをプログラム内部で処理するために用意されています。
<行番号> を 0 にした場合は通常のエラー処理となります。

[ 参照 ]
RESUME・ERL・ERR・ERROR
