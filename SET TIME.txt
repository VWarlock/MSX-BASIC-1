SET TIME
-----------------------------------------------------------
MSX2 以降
MSXPLAYer では動作しません

[ 機能 ]
時間を設定します。

[ 書式 ]
SET TIME <時間> [,A]

[ 文例 ]
SET TIME "11:24:00"

[ 解説 ]
<時間> は <時>:<分>:<秒> で、<時> <分> <秒> はそれぞれ2桁で指定します。また、<時> は24時間表記となります。午前1時2分3秒であれば 01:02:03 となります。
,A をつけた場合はアラーム時間の設定を行ないます。
MSXPLAYer では使用しているパソコン（Windows）の時間がそのまま使われます。

[ 参照 ]
SET DATE・GET DATE・GET TIME・TIME
