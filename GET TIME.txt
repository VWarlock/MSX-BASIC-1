GET TIME
-----------------------------------------------------------
MSX2 以降

[ 機能 ]
現在の時間を変数に入れます。

[ 書式 ]
GET TIME <変数名>[.A]

[ 文例 ]
GET TIME T$

[ 解説 ]
<変数名> は文字型です。得られる値は <時>:<分>:<秒> でそれぞれ2桁になります。(9以下は 0 が頭に付加されます。）<時> は24時間表記です。
,A をつけた場合は設定されているアラームの時間を取得します。
MSX PLAYer では Windows パソコンの現在の時間が取得されます。

[ 参照 ]
SET TIME・SET DATE・GET DATE・TIME
