IF GOTO ELSE
-----------------------------------------------------------
[ 機能 ]
条件判断に応じて指定行へ飛びます。

[ 書式 ]
IF <条件式> GOTO <行番号> [ELSE {<行番号>|<文>}]

[ 文例 ]
IF I>0 GOTO 30 ELSE I=I-1:GOTO 100

[ 解説 ]
<条件式> を判断し、正しい場合(0以外の場合)は GOTO 以降の <行番号> へ飛ばし、異なっている場合(0の場合)は ELSE 以降の <文> を実行します。
ELSE 後も <行番号> へ飛ばす場合は GOTO を抜いて ELSE <行番号> という形にできます。
<条件式> を判断し、正しい場合(0以外の場合)の判断を文で処理する場合は GOTO ではなく THEN を用います。
ELSE が省略されている場合は、次の文に制御が移ります。
