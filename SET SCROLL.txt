SET SCROLL
-----------------------------------------------------------
MSX2+ 以降

[ 機能 ]
ハードウェアスクロールを行ないます。

[ 書式 ]
SET SCROLL [<横ドット>][,<縦ドット>][,<マスクモード>][,<ページモード>]

[ 文例 ]
SET PAGE 50,100,1,1

[ 解説 ]
<横ドット> は画面左へスクロール量を0〜511で指定します。
<縦ドット> は画面上へスクロール量を0〜255で指定します。
それぞれ初期値は 0 ですが、省略時は最後に指定された値を保持します。
<マスクモード> は左8ドットを非表示にします。
マスクしたところを線画する事で線画処理を見せなくする効果があります。
<ページモード> の値は以下の通りです。

	0	1ページによるスクロール
	1	連続する2ページを横に連結してのスクロール
		（ページは 0 と 1、2 と 3）
