SET PROMPT
-----------------------------------------------------------
MSX2 以降
MSXPLAYer では動作しません

[ 機能 ]
プロンプトを設定します。

[ 書式 ]
SET PROMPT <プロンプト>

[ 文例 ]
SET PROMPT "READY"

[ 解説 ]
プロンプトを変更します。初期値は Ok で、MSX(1) では Ok で固定です。
<プロンプト>は6文字以内です。
この設定は MSX 本体では SRAM に記録されます。SET PASSWORD・SET TITLE と共有しているため、これらと含めて最後に設定されたものが有効となります。
初期状態は SET TITLE "" として下さい。
