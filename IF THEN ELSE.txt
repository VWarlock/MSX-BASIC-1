IF THEN ELSE
-----------------------------------------------------------
[ 機能 ]
条件判断に応じて処理を行ないます。

[ 書式 ]
IF <条件式> THEN {<行番号>|<文>} [ELSE {<行番号>|<文>}]

[ 文例 ]
IF I>0 THEN 30 ELSE I=I-1:GOTO 100

[ 解説 ]
<条件式> を判断し、正しい場合(0以外の場合)は THEN 以降の <文> を、異なっている場合(0の場合)は ELSE 以降の <文> を実行します。行番号へ飛ばす場合は GOTO を抜いて <行番号> のみにできます。また THEN の後が <行番号> である場合は THEN を GOTO に変更する事もできます。
THEN の後 に記載がない場合や ELSE が省略されている場合は、次の文に制御が移ります。
