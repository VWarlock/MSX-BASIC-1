SET BEEP
-----------------------------------------------------------
MSX2 以降

[ 機能 ]
BEEP の音色・大きさを設定します。

[ 書式 ]
SET BEEP [<音色>][,<音量>]

[ 文例 ]
SET BEEP 2,2

[ 解説 ]
この設定は MSX 本体で SRAM に記録されます。
<音色> <音量> は共に 1～4 です。
<音色> は以下の意味となります。

   1 ピッ (初期)
   2 ポーン
   3 ピーンポーン
   4 ポピプッ

<音量> は 1 が最小、4 が最大となります。
