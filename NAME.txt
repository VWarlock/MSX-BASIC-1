NAME
-----------------------------------------------------------
DiskBASIC

[ 機能 ]
ファイルのファイル名を変更します。

[ 書式 ]
NAME <旧ファイル名> AS <新ファイル名>

[ 文例 ]
COPY "AUTOEXEC.BAS" AS "AUTOEXEC.BAK"

[ 解説 ]
<旧ファイル名> <新ファイル名> はワイルドカートを用いる事ができます。
<旧ファイル名> にはドライブ名・ディレクトリ名(DiskBASIC2.0以降)を含むことができます。
