SET TITLE
-----------------------------------------------------------
MSX2 以降
MSXPLAYer では動作しません

[ 機能 ]
初期画面のタイトル・色を指定します。

[ 書式 ]
SET TITLE<タイトル>[,<色番号>]

[ 文例 ]
SET TITLE "MSX?"

[ 解説 ]
<タイトル> は6文字以内です。6文字の場合はタイトル表示で何かキーを押すまで停止します。
<色番号> は 1〜4 で、以下の色を持ちます。

	1	青
	2	緑
	3	赤
	4	オレンジ

初期値は 1 です。

この設定は MSX 本体では SRAM に記録されます。SET PASSWORD・SET PROMPT と共有しているため、これらと含めて最後に設定されたものが有効となります。
初期状態は SET TITLE "" として下さい。

[ 参照 ]
SET PASSWORD・SET PROMPT
