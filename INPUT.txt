INPUT
-----------------------------------------------------------
[ 機能 ]
キーボード・ファイルから入力されたデータを変数に入れます。

[ 書式 ]
INPUT { [<プロンプト>;] | #<ファイル番号> }<変数名>[,<変数名> ...]

[ 文例 ]
INPUT "anata no onamae nanntehno";N$

[ 解説 ]
<ファイル番号> を入れない場合はキーボードからの入力を受け付けます。
<プロンプト> があればそれを表示し、? を後につけてカーソルを表示し、入力を促します。入力された内容が <変数名> に入ります。
複数の <変数名> がある場合は , で区切って入力する事で一括して変数に入れる事ができます。一つだけ入力された場合は ?? と表示され、次の変数への入力を促します。
グラフィック画面（SCREEN 2〜8・10〜12）の場合はテキスト画面に戻ります。
なお、入力で空白や , を含む場合は " で囲む事で受け付けされます。

<ファイル番号> が入っている場合は、そのファイルから入力を受け付けます。
<ファイル番号> のファイルは OPEN により INPUT モードで開かれている必要があります。
ファイル中では、空白・改行・ , 以外の文字がデータの区切りと成ります。また、" で囲まれたものはそれで一つのデータと見なされます。255文字までを一度に読み込みます。

[ 参照 ]
OPEN・CLOSE・LINE INPUT・INPUT$
