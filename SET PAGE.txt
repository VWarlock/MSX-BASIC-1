SET PAGE
-----------------------------------------------------------
MSX2 以降

[ 機能 ]
使用する画面のページを指定します。

[ 書式 ]
SET PAGE [<表示ページ>][,<線画ページ>]

[ 文例 ]
SET PAGE 0,1

[ 解説 ]
<表示ページ> <線画ページ> はスクリーンモードと VRAM の容量によって異なります。

	SCREEN 0〜4		使用不可
	SCREEN 5〜6		VRAM 64K 0〜1
				VRAM128K 0〜3
	SCREEM 7〜8・10〜12	VRAM 64K 使用不可
				VRAM128K 0〜1

インターレースやハードウェアスクロールを行なう場合は奇数ページを <表示ページ> に指定します。
